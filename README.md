# Asuka.Net-Api

ASP.NET Web API backend for Asuka.Net Discord bot written in C# and .NET 6.

## Application
* Validation (FluentValidator)
* Mapping (AutoMapper)
* Dependency Injection
* Database (Entity Framework Core)
* MediatR

## Domain
* Models
* Entities
* Enums
* Events
* Exceptions
* Objects

## Infrastructure
* Authorization (Identity)
* Database Provider (MySQL)

## WebApi
* Swagger
* Build
* Entry
